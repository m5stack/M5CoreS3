# M5GFX

Graphics library for M5Stack series  

Supported framework  
----------------
- ESP-IDF
- Arduino for ESP32


Supported device  
----------------
- M5Stack ( Basic / Gray / GO / Fire )
- M5Stack Core2
- M5Stack CoreInk
- M5Stick C
- M5Stick C Plus
- M5Paper
- M5Tough
- M5Station
- M5ATOMS3
- UnitOLED
- UnitLCD
- [UnitRCA / ModuleRCA](docs/UnitRCA.md)
- [UnitGLASS](docs/UnitGLASS.md)
- [AtomDisplay](docs/ATOMDisplay.md) / ModuleDisplay
